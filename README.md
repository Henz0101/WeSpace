# WeSpace
This repository contains all the files for WeSpace.

## What is WeSpace?

**WeSpace** is a social networking platform, where people can create, share, and connect with their friends.

## Who built WeSpace?

**WeSpace** WeSpace is being created by a small team of students, passionately collaborating on this side project.
